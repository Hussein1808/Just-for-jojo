# 🌸 For Jojo — A Garden That Blooms For You

A little animated web garden, grown just for Jojo.

Open `index.html` in any browser, or view it live on GitHub Pages.

## What's inside
- A hero garden where flowers **grow and bloom** from the ground
- Continuous **falling petals** (canvas particle system)
- A hand-tied **SVG bouquet** — tap a flower to burst petals
- A **typewriter letter**
- The old **"loves you / loves you not"** game — rigged, it always lands on *loves you* 💕
- A **"Send me the flowers"** button that rains blossoms across the screen
- Click **anywhere** to release petals and floating hearts
- Optional gentle **ambient chimes** (off by default, 🔈 in the footer)

## Tech
Pure HTML / CSS / vanilla JS. No build step, no dependencies. Respects `prefers-reduced-motion`.
